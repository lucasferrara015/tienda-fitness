# FitShop Frontend

Tienda fitness en línea - Frontend React.

## Estado Actual: MVP

- Catálogo de productos consumido desde API REST.
- Filtros por categoría y búsqueda.
- Carrito de compras con persistencia en localStorage.
- Diseño responsive con Bootstrap.

## Tecnologías

- React 18 + TypeScript
- Bootstrap 5
- Context API para el carrito
- Fetch API para consumir backend

## Instalación

1. Clona el repositorio
2. `npm install`
3. Copia `.env.example` a `.env` y ajusta la URL de la API si es necesario.
4. `npm start`
